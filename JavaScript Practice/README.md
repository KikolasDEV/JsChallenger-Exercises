# JavaScript Practice

Bloque de práctica de JavaScript (JS Challenger) orientado a consolidar soltura resolviendo ejercicios de lógica y transformación de datos.  
Objetivo: reforzar fundamentos y patrones reales de código (strings, números, arrays, objetos, fechas, sets) con soluciones claras y documentación por bloque.

> Este bloque es previo a TypeScript y proyectos

## 🎯 Objetivos del bloque
- Practicar resolución de problemas de dificultad creciente con JavaScript.
- Dominar métodos estándar de manipulación de datos (`slice`, `filter`, `map`, `reduce`, `Object.*`, `Date`, `Set`).
- Mejorar legibilidad y consistencia: soluciones simples, directas y reproducibles.
- Documentar por bloques: resumen de lo trabajado e ideas clave.

## 🔗 Recurso
- JS Challenger: https://jschallenger.com/dashboard/

## ✅ Estándar de calidad
Cada bloque debe incluir:
- Código claro y legible (sin “magia” innecesaria).
- Uso del método/patrón que el ejercicio pide (cuando aplica).
- README del bloque con resumen técnico e ideas clave.

> Regla interna: si está “hecho” pero no se entiende, no cuenta.
